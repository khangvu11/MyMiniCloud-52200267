Hệ thống thực hành Cloud

System Health of 52200267 – cổng vào các dịch vụ trong MiniCloud.

Đây là web frontend chính (phần 2) chạy trên Nginx. Từ đây có thể truy cập API sinh viên qua Gateway (phần 9), backend ứng dụng (phần 3), cùng hệ thống giám sát & định danh.

Toàn bộ container dùng mạng nội bộ cloud-net
docker-compose · restart: unless-stopped

Quick start

# 1. Khởi động toàn bộ stack
docker compose up -d

# 2. Truy cập qua Gateway Nginx
http://localhost/
http://localhost/student/  # API /student qua gateway
Reverse proxy: api-gateway-proxy-server sử dụng Nginx, route /, /api/, /student/, /auth/.

Student Service Parts 3 & 9

Danh sách sinh viên được đọc từ database MariaDB, expose qua application-backend-server và đi qua API Gateway.

  • API gốc backend: http://application-backend-server:8081/student
  • Route gateway: http://localhost/student/
  • Giao diện: student.html (fetch JSON và render bảng)

  • Node metrics: http://monitoring-node-exporter-server:9100/metrics

Identity & API Gateway Parts 4 & 9

Keycloak quản lý người dùng & phân quyền. API Gateway sử dụng Nginx reverse proxy để route đến Web, Backend và Identity Server.

  • Keycloak: authentication-identity-server qua route /auth/
  • Gateway: api-gateway-proxy-server lắng nghe cổng 80
  • Thuật toán Round Robin cho 2 web server (phần 10)
